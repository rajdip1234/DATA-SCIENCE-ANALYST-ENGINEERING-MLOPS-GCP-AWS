Build a machine learning model to predict if a credit card application will get approved.
Project Description
Commercial banks process numerous credit card applications and often need to reject them. In this project, you'll use supervised machine learning techniques to 
automate this process, making it an efficient and cost-effective solution for banks.


Project Instructions
Use supervised learning techniques to automate the credit card approval process for banks.

Preproccess the data and apply supervised learning techniques to find the best model and parameters for the job. Save the accuracy score from your best model 
as a numeric variable, best_score. Aim for an accuracy score of at least 0.75. The target variable is the last column of the DataFrame.
